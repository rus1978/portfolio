# My Web Projects Archive

This is a collection of my older web projects (HTML, CSS, JS, PHP).  
All websites were fully developed by me — both frontend and backend — between 2010 and 2019.  
Some of the original domains are still online, but the websites have since been redesigned and are no longer my versions.
That's why this repository includes archived copies and preview images for reference.
